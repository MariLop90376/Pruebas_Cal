# Pruebas_Cal
Pruebas para verificar operaciones en calculadora

Este proyecto consiste en el desarrollo de una calculadora en Python que incluye operaciones básicas y avanzadas.  
Además, se implementaron pruebas unitarias utilizando la biblioteca `unittest` para validar el correcto funcionamiento de cada función.

La calculadora incluye las siguientes operaciones:

-  Suma  
-  Resta  
-  Producto (multiplicación)  
-  División  
-  Potencia de 2  
-  Potencia de n  
- Raíz cuadrada  

Se realizaron al menos tres pruebas por cada función, considerando:

- Números negativos   
- Números grandes  
- Caso de raíz de números negativos
